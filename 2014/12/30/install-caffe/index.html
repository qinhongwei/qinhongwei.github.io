Ubuntu 14.04.1 64bit 环境
Windows 64bit (7, 8, server 2012) 环境

Windows 环境

Windows 环境很容易配置，请参考 NUS 的 Dr. Niu Zhiheng 在 Github 上的 fork. 不过它的问题是从 Aut. 2014 以来都没更新了，不如 Caffe 的master分支新。新的功能和改进，没有加进去，不过作为上手，应该是最快的。
备注：Dr. Niu Zhiheng 所在的组是 Yan Shuicheng 的组。上次去新加坡对这个组略有接触，这个组用 Caffe 挺多的，人挺 nice。

Linux 环境(以 Ubuntu 14.04 为例)

Ubuntu 环境也很容易配置，请直接参考 Caffe 官方 installation guide.
另外，官方没有细说 CUDA 和cuDNN的安装。节省精力起见，可参考：

  • Caffe + Ubuntu 14.04 64bit + CUDA6.5 配置说明
  • 以上帖子的另一个版本

注意事项

  1. 注意这两个帖子的差别：CUDA的两种安装方式将对应于不同安装文件夹名称. deb安装方式对应于 cuda, 而run 安装方式对应于cuda-6.5，这涉及到多个地方的设置，要多加注意。
  2. 其他地方没有太多要注意的，编译 Caffe 我遇到过一个小问题，按照出错提示操作就解决了。
  3. 多核编译的命令：sudo make all -j24这台机器 2 块 12 核的 CPU，总共 24 核心，48 线程。几秒钟就编译好了。

总之，安装 Caffe 比在苛刻的环境下安装 Ubuntu 简单多了。

八卦

关于 Caffe, Caffe 第一版的作者 Yangqing JIA, Windows porting 作者 Zhiheng Niu, NUS, UC Berkeley, Tsinghua 更是有微妙的关联。而这个圈子，我估计必是 Lecun 等人看不上的，暗暗较劲儿的。学术圈的八卦，真是好不热闹。这个话题，也许可以改日多加八卦。

如果觉得帮到了你，打赏一下吧！
